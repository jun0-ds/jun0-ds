### Featured Projects

- **[sonmat](https://github.com/jun0-ds/sonmat)** — A Claude Code plugin that builds verification and discipline habits into AI-human collaboration. - **[bobusang](https://github.com/jun0-ds/bobusang)** — A multi-device memory system for Claude Code. Syncs context across Windows, WSL2, and Linux via git-based auto-sync, so switching machines doesn't reset your AI assistant to zero.
